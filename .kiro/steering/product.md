# Product Overview

Ragzy AI Engage is an AI-native customer engagement platform that provides personalized chatbot functionality trained on website content. The platform enables businesses to instantly answer visitor questions through an intelligent chatbot widget.

## Core Features

- AI-powered chatbot widget for websites
- Knowledge base management with vector search (Qdrant)
- Review form system for customer feedback
- Team collaboration with workspace management
- Real-time conversation tracking and analytics
- Email notification system for team updates
- Subscription-based access control

## Key Components

- **Frontend**: Customer-facing dashboard and widget embedding
- **Backend**: FastAPI service with RAG (Retrieval-Augmented Generation)
- **AI Integration**: OpenRouter API with Grok model for cost-effective responses
- **Vector Database**: Qdrant for semantic search and knowledge retrieval
- **Authentication**: Firebase Auth with workspace-based access control
- **Database**: Firestore for user data, conversations, and business logic
